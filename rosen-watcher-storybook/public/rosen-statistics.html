This page shows example usage of the <rosen-watcher-component> tag using watchers component. This is a javascript component that can be used in any html page (it is angular under the hood, but works with any html + js).
The latest JavaScript versions and styles are used on this page. Download these files and use them similarly (see source of this html page).
For detailed usage, see the documentation .

  • styles-6DQTSDIR.css (This file can be downloaded and used if the default styling is something one wants to build on, the file is not necessary and if all styling is to be done custom this file can be ignored.)
  • rosen-watcher-components.3.48.6.js (Required)
